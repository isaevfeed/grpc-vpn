# SOCKS5 Proxy service

Проект подразумевался как элементарное VPN чисто в учебных целях, но в процессе решил оставить
реализацию прокси SOCKS5 без общей оберкти. Работает чисто экспериментально и не рекумендуется
для постоянно использования

### Запуск

`make run-vpn-server` - запуск сервера

`make run-vpn-client` - запуск клиента

Далее можно протестировать работу SOCKS5

`curl --socks5 {server_host}:{server_port} {domain/IP}`
